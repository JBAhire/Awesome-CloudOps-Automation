<h2>Update Dispute</h2>

<br>

## Description
This Lego Update Dispute.


## Lego Details

    stripe_update_dispute(handle: object, dispute_id:str, submit:bool,metadata=None, evidence)

        handle: Object of type unSkript stripe Connector
        dispute_id: Dispute Id
        submit: Whether to immediately submit evidence to the bank.
        metadata: This can be useful for storing additional information about the object in a structured format.
        evidence: Evidence to upload, to respond to a dispute.


## Lego Input
This Lego take nine input handle, dispute_id, submit, metadata and evidence.
